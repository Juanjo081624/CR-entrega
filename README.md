# Novedades Turno CR / Rescate CUA

App web/mobile responsive para completar novedades operativas del turno CR / Rescate CUA y generar un mensaje listo para WhatsApp.

## Archivos

- `index.html`: aplicación completa con HTML, CSS y JavaScript.
- `manifest.json`: configuración básica para uso tipo PWA.
- `service-worker.js`: caché básico para funcionamiento offline del formulario.
- `icon-192.png` y `icon-512.png`: íconos de la app.

## Cómo publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube estos archivos en la raíz del repositorio.
3. Ve a `Settings`.
4. Entra a `Pages`.
5. En `Build and deployment`, elige:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Guarda.
7. Abre la URL publicada por GitHub Pages.

## Uso

Completa los campos, revisa la vista previa, copia el mensaje o abre WhatsApp.
